Nook - パーソナル情報ハブ 📚

2025年3月作成

📌 プロジェクト概要

動詞さんが開発されたNookにしています。公開に感謝いたします。

Nookは、さまざまな情報ソース（Reddit、Hacker News、GitHub Trending、Tech Feed、arXiv論文）からコンテンツを収集し、一元的に表示するパーソナル情報ハブです。

完全にローカルで動作！
取得した記事はすべて日本語に変換！

🔍 主な機能

  • 複数の情報ソースからのコンテンツ収集と表示
  • 日付ごとのコンテンツフィルタリング
  • 天気情報の表示（OpenWeatherMap API）
  • Grok APIによるテキスト生成・要約

🌐 情報ソース一覧：

  • 📱
    Reddit人気投稿
  • 📰
    Hacker News記事
  • 💻
    GitHub Trendingリポジトリ
  • 🌐
    技術ブログのRSSフィード
  • 📄
    arXiv論文

🏗️ アーキテクチャ

フロントエンド（React + Vite）

サイドバー（ソース選択、日付選択、天気表示）とコンテンツビューア

バックエンド（FastAPI）

コンテンツAPI

サービス

  • Reddit Explorer
  • Hacker News Retriever
  • GitHub Trending
  • Tech Feed
  • Paper Summarizer
